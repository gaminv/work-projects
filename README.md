# 📌 Full Stack Telegram-бот  
## Генератор персональных Valentine-сайтов

🔗 Бот: https://t.me/make_valentine_bot  

Telegram-бот, автоматически генерирующий персональный Valentine-сайт по шаблону.  
Пользователь вводит текст, загружает изображение — система создаёт готовую веб-страницу и размещает её на сервере.

---

## 🧩 Реализованный функционал

- Генерация сайта по шаблону через Telegram-бота
- Динамическая подстановка текста и изображения
- Реферальная система
- Админ-панель
- Встроенный чат поддержки
- Интеграция ЮKassa (временно отключена)
- Автоматическая генерация чеков через «Мой налог»
- Пользовательское соглашение
- Деплой и серверная инфраструктура
